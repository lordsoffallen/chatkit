# @chatkit/cli

Source installer for Chatkit UI blocks.

## Install

Use directly with `npx`:

```bash
npx @chatkit/cli add artifact document
```

## Current Commands

```bash
npx @chatkit/cli add artifact document
```

This installs source files into:

```text
components/chatkit/
```

And merges the required dependencies into the target app's `package.json`.

Implemented now:
